## PART 1: FREE KEYWORD RESEARCH TOOLS

### 1. Google Search Console (100% FREE, Most Important)

**What it does:** Shows ACTUAL keywords your site ranks for + impressions/clicks

**How to use:**

```
1. Go to: https://search.google.com/search-console
2. Add your website (verify ownership)
3. Navigate to: Performance > Search results
4. See:
   - Keywords you rank for
   - Average position
   - Impressions (how many times you appeared)
   - Clicks (how many clicked through)
   - CTR (click-through rate)
```

**Key insights:**

a) **Low-hanging fruit:** Keywords where you rank #11-20 (page 2)
   - These are easiest to move to page 1
   - Filter by: Position 11-20
   - Focus on improving these pages first

b) **High impression, low clicks:** Good position but bad CTR
   - Position 1-10 but CTR < 5%
   - Fix: Improve meta title/description
   - Make them more compelling

c) **Content opportunities:** Keywords you rank #5-10 for
   - Already getting some traction
   - Add more content to these pages
   - Could easily rank #1-3

**Export data:**
```
1. Click "Export" button
2. Download as Google Sheets or Excel
3. Sort by impressions (high to low)
4. These are your opportunities!
```

### 2. Google Keyword Planner (FREE with Google Ads account)

**What it does:** Shows search volume, competition, related keywords

**Setup (FREE, no ad spend required):**

```
1. Go to: https://ads.google.com/
2. Create free account (don't run any ads!)
3. Tools & Settings > Planning > Keyword Planner
4. Click "Discover new keywords"
```

**How to find keywords:**

**Method 1: Start with a seed keyword**
```
1. Enter: "legal notice India"
2. Click "Get results"
3. See:
   - Search volume (monthly searches)
   - Competition (Low/Medium/High)
   - Related keywords (hundreds of suggestions)
```

**Method 2: Analyze competitor's website**
```
1. Click "Start with a website"
2. Enter competitor URL: vakilsearch.com/legal-notice
3. Google shows keywords that page ranks for
4. Filter by: 1K-10K monthly searches
5. Export as CSV
```

**Pro Tips:**
- Don't run ads! Just use it for research (100% free)
- Focus on keywords with "Low" competition
- Target 1,000-10,000 monthly searches
- Ignore branded keywords (like "VakilSearch")
